# Jasiri Gym

**Book Bold. Train Strong.**

## Description
Jasiri Gym is a responsive multi-page booking website where users can browse fitness classes and book a session online. 

## Features
- Browse available gym classes
- Filter classes by type(Yoga, Cardio,Strength, Boxing, Dance)
- Book a session through an online form
- View booking confirmation with full details
- Exercise library powered by the wger Workout Manager API

## Built with
- HTML
- JavaScript
- Vanilla CSS
- [wger Workout Manager API](https://wger.de/api/v2/)

## Pages
- `index.html` - Homepage
- `classes.html` - Classes and Exercise Library
- `booking.html` - Booking Form
- `confirmation.html` - Booking Confirmation
